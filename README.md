# Week 1: HTML Foundations & Semantic Structures
Objective: Learn HTML structure, semantic tags, and basic page building.
Scenario: You are applying for a front-end role, and the hiring challenge is to design your profile and showcase details.
Tasks:
1. Create a personal portfolio page using semantic HTML.
   - Include photo, bio, education, skills, and contact info.
   - Use figure and figcaption for the profile image.
   - Output: Clean, semantic HTML layout. 
2. Create a timetable using table.
   - Use thead, tbody, tfoot.  
   - Output: Neat academic timetable.
